# Panduan Kontribusi OURIN MD

Terima kasih atas minatmu untuk berkontribusi di **OURIN MD V2.3.0**! 

## Cara Berkontribusi

1. Lakukan *Fork* pada repository ini
2. Buat *Branch* baru (`git checkout -b fitur-kerenmu`)
3. Lakukan pengubahan (*Commit* perubahanmu)
4. Pastikan kodemu berjalan dengan lancar tanpa error (`npm start`)
5. Dorong *Branch* ke GitHub (`git push origin fitur-kerenmu`)
6. Buat *Pull Request* ke repository ini

## Aturan Penulisan Kode
- Wajib menggunakan sintaks JavaScript ES/CommonJS yang modern dan terstruktur.
- Jangan menyertakan *session* WhatsApp (folder `data/`) pada *Pull Request*.
- Dilarang keras menaruh / mem-publish *API Keys* (seperti Google, Groq, dll) di dalam kode sumber repositori umum.
